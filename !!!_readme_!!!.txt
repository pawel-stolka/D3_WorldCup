how to set local server with json file?

1. Install Node.js
2. In command line run 'npm init'
	- several question, just hit enter except call app as 'app.js' 
3. run 'npm install express --save'
3. run 'npm install cors --save'
4. In your packages.json under "scripts" append line 
	"start": node app.js"
5. Run 'npm start'
	That's it :)
